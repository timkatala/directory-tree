# Directory Tree

## Features
* [CI: GitHub Actions](.github/workflows/code_check.yml)
* Code checks
  * `flake8`
  * `black`
* [Tests (`Pytest`)](tests)

## Usage
* Install dependencies
  * `pip install -r requirements.txt`
* Run
  * Modify `input.txt` file with inputs
  * `python directories.py`
* Tests
  * `pytest .`
